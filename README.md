

# Pokémon RestfulAPI By CakePHP3


## 環境構築

```
// CakePHP3インストール(xxxはプロジェクト名)
composer create-project --prefer-dist cakephp/app xxx

// PHPUnitインストール
composer require --dev phpunit/phpunit:"^5.7|^6.0"

// Goutteインストール
composer require fabpot/goutte
```
